# ROS2 Learning Journey 🤖

This repository documents my journey learning ROS2 (Robot Operating System 2) as I build skills in robotics, autonomy, and real-time systems.

## 📌 Focus Areas
- ROS2 basics (nodes, topics, services, actions)
- Publisher / Subscriber communication
- TF2 transforms
- Launch files
- Sensor integration (camera, LiDAR, IMU)
- Simulation (Gazebo / Isaac Sim)
- Robot navigation basics

## 🎯 Goal
The goal of this repo is to:
- Understand how ROS2 works under the hood
- Build real robotic systems using modular components
- Prepare for work in robotics, autonomous systems, and aerospace applications
- Combine ROS2 with computer vision and AI (YOLO, OpenCV, etc.)

## 🛠️ Tools & Technologies
- ROS2 (Humble / Iron)
- Python & C++
- Gazebo / Ignition
- Isaac Sim (NVIDIA)
- OpenCV
- RViz2
